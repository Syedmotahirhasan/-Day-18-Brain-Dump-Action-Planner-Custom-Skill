# Day-18-Brain-Dump-Action-Planner-Custom-Skill

Skill
📌 Project Overview

This project demonstrates the creation of a Brain Dump Action Planner as a Claude Custom Skill. The skill is designed to transform unstructured notes—such as meeting notes, brainstorming sessions, class notes, voice memo transcripts, and project discussions—into organized, actionable insights.

By leveraging Claude's Custom Skills feature, the planner automatically extracts summaries, action items, risks, blockers, deadlines, and open questions without requiring users to repeatedly provide the same instructions. The output is presented through an interactive dashboard that makes information easier to review, prioritize, and act upon.

🎯 Project Objectives
Create a reusable Claude Custom Skill named brain-dump-action-planner.
Convert raw notes into structured summaries.
Automatically identify action items and assign priorities.
Detect risks, blockers, and unresolved questions.
Generate an interactive dashboard for visualizing results.
Demonstrate how Custom Skills improve productivity and workflow automation.
Document the implementation process and key learnings.
